# SkyQuest 🚀🔭

SkyQuest is a web application designed for amateur and professional astronomers to track celestial objects, log observations, and mark which ones have been seen. Initially focused on the Messier catalog, it will expand to include other deep-sky catalogs in the future.

## Features 🌌

- 📌 **Messier Object Tracking** – Browse and track objects from the Messier catalog.  
- ✅ **Observation Log** – Mark objects as observed and add personal notes.  
- 🔎 **Search & Filter** – Easily find objects by name, type, or visibility.  
- 🌍 **Expandable Catalogs** – Future support for other astronomical catalogs like NGC.  

## Tech Stack 🛠️

- **Frontend**: React (Next.js) with TypeScript  
- **Backend**: Spring Boot (if using Java)  
- **Database**: PostgreSQL
- **Deployment**: Docker
